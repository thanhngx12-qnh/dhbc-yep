Cấu hình

Quản lý Database (database.json):

Hệ thống sẽ tự động tìm file database.json cùng thư mục.

  • Nếu có file: Dữ liệu sẽ được load từ file.
  • Nếu không có file (hoặc bị chặn): Sẽ dùng dữ liệu LocalStorage.
  • Lưu ý: Chỉnh sửa/Thêm/Xóa ở đây CHỈ lưu vào bộ nhớ tạm. Để cập nhật file database.json, hãy dùng nút "Xuất File" bên dưới và ghi đè file cũ.
